# Instructions

- Keep changes scoped to the requested task.
- Use deterministic, non-interactive commands.
- Use `fd` for file discovery and `rg` for text search.
- Use `ast-grep` for syntax-aware code search when available.
- Use `jq` for JSON and `yq` for YAML/XML.
- `dev_docs` documents must use `YYYYMMDD-HHmm-name.md` filenames.
